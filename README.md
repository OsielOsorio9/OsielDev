# Portafolio Profesional - Osiel Osorio

Este es mi portafolio profesional como Desarrollador Web e Ingeniero en Telecomunicaciones.

## Características
- 🚀 **Tecnologías:** HTML5, CSS3 (TailwindCSS), JavaScript.
- 🌓 **Modo Oscuro/Claro:** Soporte nativo con persistencia en el navegador.
- 🌍 **Multilenguaje:** Disponible en Español e Inglés.
- 📱 **Responsive:** Optimizado para todos los dispositivos.
- ✍️ **Blog:** Sección de artículos para resolver dudas frecuentes sobre desarrollo web.

## Estructura del Proyecto
- `index.html`: Página principal con secciones de Hero, Servicios, Sobre Mí y Contacto.
- `blog.html`: Listado de artículos del blog.
- `article.html`: Plantilla para lectura de artículos individuales.
- `css/styles.css`: Estilos personalizados y animaciones.
- `js/main.js`: Lógica del sitio (idioma, tema, menú móvil).

## Despliegue en GitHub Pages
1. Sube estos archivos a un repositorio en GitHub.
2. Ve a **Settings** -> **Pages**.
3. Selecciona la rama `main` y la carpeta `/root`.
4. ¡Listo! Tu sitio estará en `https://OsielOsorio9.github.io/nombre-del-repo/`.
